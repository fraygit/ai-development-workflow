## Local dev prerequisites

Before any story tasks:
1. Complete **Task 1.1** (Node 20, Docker Compose, PostgreSQL, Redis, env vars, GitFlow branches, Vitest).
2. Complete **Task 1.3** (Clerk dev app + webhook config).
3. The backend API runs on `localhost:3001`. The web app runs on `localhost:3000`.

**Branching convention:** every task is implemented on `feature/<task-id>-<short-title>` branched from `develop`, and merged back to `develop` via PR before dependent tasks begin. See Task 1.1 for the full GitFlow rules.

**Test runner:** Vitest for all backend and frontend unit/integration tests. Run `npx vitest run` before raising a PR. Integration tests require a running `aidevflow_test` PostgreSQL database.
